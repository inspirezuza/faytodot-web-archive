## Ultimate Tailwind CSS Tutorial // Build a Discord-inspired Animated Navbar

[Fireship Tailwind Tutorial](https://fireship.io/lessons/tailwind-tutorial/)

add this code `tailwind.config.ts` for custom colors

```
<!-- tailwind.config.ts -->
theme: {
  extend: {
    colors: {
      primary: "#202225",
      secondary: "#5865f2",
    },
  },
},
```
